        # Attributes

        参数 说明 类型 可选值 默认值
        hotList 商品详情热门商品列表 Array - 无
        img 热门商品列表item商品图片 string - 无
        name 热门商品列表item商品名称 string - 无
        price 热门商品列表item商品价格 string - 无
        type 商品列表展示方式 type='up'为竖行展示，默认为横行 string up 无
        num 当前展示的轮播图个数 Number - 3
